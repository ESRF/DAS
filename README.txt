The Data Analysis Server (DAS) project

Authors: Staffan Ohlsson and Olof Svensson

Copyright: European Synchrotron Radiation Facility

The Data Analysis Server is a TANGO server which acts as a "mailbox" 
between the beamline control application (e.g. mxCuBE/BsxCuBE) and the 
EDNA/workflow TANGO servers.

For more detailed documentation see src/doc_html/index.html
